# goceries

Shopping list app built on the same technical foundation as `fundApp`.

## Stack

- Vue 3
- Vite
- TypeScript
- Pinia
- Vue Router
- Supabase
- Tailwind CSS
- Sass
- ESLint + Oxlint + Prettier

## Setup

```sh
yarn
cp .env.example .env
```

Fill in:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Commands

```sh
yarn dev
yarn build
yarn lint
yarn migrate
```
